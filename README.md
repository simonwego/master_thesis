# Inference and Identifiability of Intransitive Dominance Models with Latent Structure

This repository contains the R code and generated analysis data used in the master’s thesis **_Inference and Identifiability of Intransitive Dominance Models with Latent Structure_**.

The project studies estimation, model comparison, identifiability, numerical stability, bootstrap inference, and conditional latent-variable diagnostics for four probabilistic dominance models fitted to dyadic interaction data from `DomArchive`:

- **LBT**: latent Bradley--Terry model;
- **LID**: latent intransitive dominance model;
- **DABT**: LBT with dyad-specific random effects;
- **DALID**: LID with dyad-specific random effects.

The models are implemented with `RTMB`. Latent variables are integrated using the Laplace approximation, while the fixed parameters are optimized with `nlminb()`.

## R scripts

### `FourModelsComparisonInference.R`

Main model-fitting and comparison script. It:

- filters and prepares count matrices from `DomArchive`;
- fits LBT, LID, DABT, and DALID using multiple starting values;
- extracts likelihoods, AIC values, parameter estimates, latent positions, dyad effects, and fitted probabilities;
- computes gradient and fixed-parameter Hessian diagnostics;
- calculates probability-agreement measures;
- saves the main model-fitting datasets used in the thesis.

### `BootstrappingOnDomArchive.R`

Parametric bootstrap script. It:

- simulates interaction matrices from fitted models;
- refits the models to bootstrap samples;
- performs bootstrap likelihood-ratio tests;
- generates parameter-bootstrap samples and summaries;
- records convergence failures and non-finite estimates.

### `LRTGridAnalysis.R`

Simulation study of likelihood-ratio statistics under the null model. It:

- simulates complete dyadic systems over selected values of group size and interactions per dyad;
- compares empirical LRT distributions with $0.5\chi^2_0+0.5\chi^2_1$;
- produces distribution summaries, histograms, QQ plots, and PP plots.

### `MCMC.R`

Conditional MCMC diagnostics for the latent variables. It:

- samples from the conditional latent distribution given the fitted fixed parameters;
- uses adaptive random-walk Metropolis--Hastings;
- computes acceptance rates, effective sample sizes, integrated autocorrelation times, skewness, and excess kurtosis;
- produces marginal-distribution plots used to assess the Gaussian approximation underlying the Laplace approximation.

## Generated analysis datasets

The `.rds` files below are generated by the scripts and constitute processed analysis data rather than the original `DomArchive` observations.

### `all_bootstrap_param_samples.rds`

Long-format dataset containing the individual parameter estimates from the parametric bootstrap.


### `bootstrap_parameters_summary.rds`

Aggregated parameter-bootstrap results derived from `all_bootstrap_param_samples.rds`.


### `four_models_comparison_dyad_effects.rds`

Fitted random effects quantities from the four model fits.

### `four_model_comparison_latent_positions.rds`

Estimated latent positions for all individuals, datasets, and models.

### `four_model_comparison_probability_agreement.rds`

Dataset-level measures comparing fitted dyadic probabilities with the empirical saturated probabilities.

### `four_model_comparison_results.rds`

Main wide-format result dataset from the four-model analysis.

This is the principal dataset used for cross-dataset comparisons, tables, regression analyses, and most summary figures in the thesis.

> The exact filenames produced by `FourModelsComparisonInference.R` depend on the value of the object `tag`. In the supplied script, files may therefore include an additional tag such as `four_models_comparison_filtered_betterdiag_` before the dataset-specific suffix.

## Recommended workflow

1. Run `FourModelsComparisonInference.R` to generate the main model fits and processed model-comparison datasets.
2. Run `BootstrappingOnDomArchive.R` to generate parameter-bootstrap samples, bootstrap summaries, and bootstrap LRT results.
3. Run `LRTGridAnalysis.R` to assess the finite-sample reference distribution of the LRT statistics.
4. Run `MCMC.R` to assess conditional latent-variable mixing and deviations from Gaussianity.

## Notes on reproducibility

- Several paths use `~` or project-relative directories and may need to be changed on another system.
- The scripts combine reusable functions with exploratory analysis code and are intended to document the thesis workflow rather than form a standalone R package.
- Bootstrap and MCMC analyses are computationally expensive. Reduce the number of replicates or iterations when testing the code.
- Reproducibility seeds are set in the main simulation sections, commonly to `123`.
